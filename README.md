# LevelUp Academy

LevelUp Academy is a lightweight, mobile-first, gamified educational web platform designed specifically for Moroccan middle school students. 

## Features
- **Math Arena:** Mental math multiplication challenges with combos and timers.
- **Language Academy:** Learn French and English vocabulary through multiple-choice questions.
- **Gamification:** Earn XP, coins, and maintain streaks to level up.
- **Funny System:** Humorous, culturally relevant feedback when mistakes are made.
- **Offline / Backendless:** Designed to run purely on client-side technologies (HTML/CSS/JS) with Firebase for optional cloud sync.

## Technology Stack
- Vanilla HTML5
- CSS3 (Custom Variables, Flexbox/Grid, Animations)
- Vanilla JavaScript (ES6+)
- Firebase (Auth & Firestore for persistence)

## How to Run Locally
1. Clone or download this repository.
2. Open `index.html` directly in any modern web browser (no build tools or local server required).
3. To test mobile responsiveness, open the Developer Tools (F12) and toggle device emulation.

## Project Structure
- `index.html`: The main SPA container.
- `css/`: Stylesheets.
- `js/`: Application logic, split into core engines, services, modules, and views.
- `data/`: JSON-like JavaScript arrays holding vocabulary.

Enjoy learning!
